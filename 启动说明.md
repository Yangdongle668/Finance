# 精斗云云会计系统 - 启动说明

## 快速启动

### 前提条件
- Node.js 18+（推荐 LTS 版本）
- npm 9+

### 第一次启动

```bash
# 1. 安装所有依赖
cd C:\Users\Yang\Desktop\Finance
npm install       # 安装根目录 concurrently
cd server && npm install
cd ../client && npm install

# 2. 初始化后端配置
cd server
copy .env.example .env

# 3. 初始化数据库（创建管理员账号 + 标准科目表 + 期间）
npm run seed

# 4. 回到根目录启动开发服务
cd ..
npm run dev
```

### 访问地址
- **前端界面**：http://localhost:5173
- **后端 API**：http://localhost:3001
- **健康检查**：http://localhost:3001/health

### 默认账号
| 用户名 | 密码      | 角色   |
|--------|-----------|--------|
| admin  | Admin@123 | 管理员 |

## 扩展开发指南

### 新增模块
1. 在 `server/src/domain/` 下新建类型文件
2. 在 `server/src/infrastructure/repositories/` 下实现数据访问
3. 在 `server/src/application/services/` 下实现业务逻辑
4. 在 `server/src/api/routes/` 下添加路由，并注册到 `server.ts`
5. 在 `client/src/pages/` 下新建页面，注册到 `App.tsx`

### 金额处理规范
- **数据库存储**：以"分"（整数）为单位，避免浮点精度问题
- **API 请求**：前端传入"元"（小数），后端自动 × 100 转换
- **API 响应**：以"元"返回（后端 / 100 后输出）

### 安全设计
- JWT 认证，7天有效期
- 角色权限：admin / supervisor / accountant / cashier / viewer
- 结账后凭证自动锁定，禁止修改
- 反结账需要 admin 权限
- 所有金额操作均有审计日志
