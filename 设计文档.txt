精斗云云会计系统 - 单账套系统设计文档
一、系统概述
1.1 系统定位
精斗云云会计是一款针对个人/小微企业的云端财务管理系统，采用单账套架构设计，支持完整的财务记录、核算、报表分析功能。
1.2 核心特点

云端部署：无需本地安装，浏览器即可访问
单账套设计：专为个人使用优化，无需多账套切换
实时协作：支持多人在线操作与权限管理
完整核算：覆盖凭证、账簿、报表全流程
智能分析：提供财务指标、行业对标等分析工具


二、核心功能模块设计
2.1 凭证管理模块（证）
功能目标：记录企业所有经济业务的会计凭证
主要功能：

新增凭证：支持手工录入凭证，包括：

基本信息：凭证号、日期、摘要
行项目：借方科目、贷方科目、金额
附件：支持扫描件或文件上传
核算项目：成本中心、部门、客户等维度


凭证查询：

按凭证号、日期、摘要搜索
按科目、核算项目筛选
按凭证状态查看（草稿、待审核、已审核、已记账）


凭证操作：

编辑修改（仅对草稿凭证）
删除凭证（已审核凭证需反向记账）
审核批准流程
批量记账、反向记账


凭证导入：

支持Excel批量导入
支持第三方系统（进销存、工资）自动生成凭证



操作流程：
新增凭证 → 录入数据 → 保存为草稿 → 提交审核 → 审核批准 → 记账入账
         ↓(修改)          ↓(驳回)
         编辑凭证          修改重新提交
数据结构：
凭证主表：
- 凭证ID、凭证号、凭证日期
- 摘要、附件说明
- 制单人、审核人、记账人
- 状态、创建时间、修改时间

凭证行表：
- 行号、科目代码、科目名称
- 方向（借/贷）、金额
- 核算项目维度值
- 备注

2.2 账簿管理模块（账簿）
功能目标：展示企业各类账簿，供查询和分析
子模块：
2.2.1 日记账（出纳日记账）

功能：记录现金和银行存款的进出情况
查看维度：

账户视图：按银行账户、现金账户分类显示
时间视图：按日期段统计
汇总视图：余额、收入、支出三角形展示


主要指标：

期初余额、期末余额
本期收入、本期支出
日均余额、日均收支



2.2.2 明细账（详细分类账）

功能：展示每个科目下的明细交易记录
查看方式：

按科目树形展开
显示借方、贷方、余额三栏
支持按期间、核算项目筛选


操作功能：

追溯凭证：点击明细可查看原始凭证
导出数据：支持Excel导出



2.2.3 总账（一级分类账）

功能：展示所有一级科目的期初、期末余额
显示内容：

科目、借方累计、贷方累计、期末余额
对比上期、上年同期
环比变化分析



2.2.4 科目余额表

功能：核心报表，展示全部科目的对账数据
数据组成：

科目代码、科目名称、科目级别
期初余额（借/贷）
本期发生额（借/贷）
期末余额（借/贷）


核算验证：

借贷平衡检查：所有借方余额 = 所有贷方余额
支持按核算项目、部门分类汇总



数据流：
凭证记账 → 科目汇总 → 科目余额表 → 财务报表
                  ↓
            明细账、总账、日记账

2.3 报表分析模块（报表）
功能目标：生成标准财务报表和分析报告
主要报表：
2.3.1 资产负债表（Balance Sheet）

作用：反映企业某一时点的财务状况
组成：

资产端：流动资产、非流动资产
负债端：流动负债、非流动负债
所有者权益：实收资本、未分配利润


功能：

按期间对比（本期、上期、去年同期）
金额和百分比同时展示
导出打印功能



2.3.2 利润表（Income Statement）

作用：反映企业某期间的经营成果
组成：

营业收入 - 营业成本 = 毛利
毛利 - 销售费用 - 管理费用 - 财务费用 = 营业利润
营业利润 ± 营业外收支 = 利润总额
利润总额 - 所得税费用 = 净利润


计算指标：

毛利率、营业利润率、净利率
费用占比
期间对比增长率



2.3.3 现金流量表

作用：反映企业现金流动情况
三大活动：

经营活动现金流
投资活动现金流
筹资活动现金流



2.3.4 自定义报表

功能：用户可自定义报表样式
功能：

选择行列科目
设置报表期间
选择数据粒度（汇总/明细）
保存报表模板



报表输出：

PDF导出
Excel导出
打印输出
邮件发送


2.4 结账管理模块（结账）
功能目标：完成月度（或期间）结账流程
结账流程：
月末核对 → 生成结转凭证 → 期末调整 → 最终审核 → 结账锁定 → 生成报表
   ↓
- 凭证核对
- 账簿平衡检查
- 对账单对接
主要步骤：

期末核对

检查未记账凭证
验证科目余额平衡
确认所有发票已入账


生成结转凭证

损益类科目结转至本年利润
本年利润结转至利润分配
利润分配结转至未分配利润


期末调整

计提各类费用（折旧、摊销等）
应收、应付确认
存货盘点调整


结账审核

财务主管审核
管理层确认


结账锁定

锁定当期数据
禁止修改历史凭证
生成财务报表快照



结账管理：

支持月度、季度、年度结账
支持结账反结（含特殊权限）
结账期间历史管理


2.5 资产管理模块（资产）
功能目标：管理固定资产生命周期
主要功能：

资产登记：

资产名称、分类、原值
取得日期、预计使用年限
资产位置、使用部门
资产编号、条码


折旧计算：

年限平均法
工作量法
加速折旧法（选配）
自动生成折旧凭证


资产变动：

增加、减少、转移
增值、减值
报废清理


资产查询：

资产卡片
折旧清单
资产统计



数据：
资产主表：资产ID、资产名称、分类、原值、折旧年限
折旧表：资产ID、月份、折旧额、累计折旧、净值
变动表：资产ID、变动类型、变动日期、变动金额

2.6 发票管理模块（发票）
功能目标：管理进项/销项发票
子功能：
2.6.1 进项发票管理

发票录入（手工/扫描）
进项税抵扣管理
发票认证状态追踪
防重复入账校验

2.6.2 销项发票管理

发票开具记录
销项税征收管理
发票作废处理
开票限额监控

2.6.3 发票统计

进项发票汇总
销项发票汇总

增值税申报数据提供


2.7 出纳管理模块（出纳）
功能目标：管理现金和银行账户
主要功能：

银行账户管理：

账户信息：账户名、账号、开户行

收付款管理：

收款单：来源、金额、用途

2.8 工资管理模块（工资）

员工档案：

基本信息：姓名、身份证、入职日期

基本工资、绩效奖金

企业所得税计算

收入成本构成

3.2 数据一致性保证
双边账制：

每张凭证必须借贷相等
每条凭证行必须指定科目和金额
科目余额借贷必须平衡

审计追踪：

记录所有操作人、操作时间
保存修改前后的版本
追踪凭证的全生命周期

数据验证：

凭证记账前检查合法性
结账前进行平衡检查
导入数据前进行格式校验

六、安全与权限设计
6.1 权限体系
角色预设：

管理员：全部功能、全部数据
财务主管：审核、结账、报表查看
会计：凭证录入、查询、打印
出纳：收付款、银行对账、现金管理
查看人员：仅数据查看权限

权限细分：
菜单权限：是否能访问某个菜单
数据权限：只能看到指定部门/项目的数据
操作权限：是否能进行增删改操作
审批权限：是否能审核、批准凭证
6.2 数据安全

加密存储：密码、敏感数据加密
审计日志：所有操作记录完整保存
备份恢复：定期自动备份，支持快速恢复
操作锁定：结账后禁止修改历史凭证
访问控制：IP限制、登录地点限制（可选）

6.3 验证机制

身份验证：用户名+密码+双因素认证
会话管理：闲置超时自动登出
操作确认：重要操作（删除、结账）需再次确认
